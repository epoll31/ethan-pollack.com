Mac Terminal Utility

$Path

'$PATH' is essentially a list of paths that your terminal session can search through to shorten common used paths.

Viewing your $PATH

Run the command 'echo $PATH' in your terminal. This will return the list of paths. They will be separated by a space, colon, etc.

Adding to $PATH

What you will need: the full command path and the shorthand command name. Let's call these '/full/path' and 'name', respectively. Run the command 'sudo sh -c 'echo "/full/path" >> /etc/paths.d/name'

Running a command

Running a command is as simple as writing the command name followed by any parameters that are required. If the command is not in your $PATH then you must specify the full path name of a command.

Helpful Commands

echo

Run the command 'echo text' in your terminal. This will repeat any input as an output.

which

Run the command 'which (insert command)' in your terminal. This will display the path of the specified command.

whoami

Run the command 'whoami' in your terminal. This will display the username of the current user.
